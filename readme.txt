Installation
============

The following extensions are required:

* karma_sphinx_theme (https://pypi.org/project/karma_sphinx_theme/)
* sphinxprettysearchresults (https://pypi.org/project/sphinxprettysearchresults/)
* sphinxcontrib.images (https://pypi.org/project/sphinxcontrib-images/)
* sphinx_sitemap (https://pypi.org/project/sphinx-sitemap/)
* sphinxcontrib-contentui (https://pypi.org/project/sphinxcontrib-contentui/, sphinxcontrib-contentui.readthedocs.io)

Detailed instructions: https://drive.google.com/open?id=1hB6-dbKdQc9jBv4kG--N2VjPKTZoaLFS

Customization
=============

Custom syles and scripts can be added to:

* _static/custom.css
* _static/custom.js



Build Docs
==========

Run make html. 

The output is stored to the ..\rr-build folder. 

To change the folder, edit Makefile and make.bat.
